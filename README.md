# 飞扬大修系统

基于 Vue 和 Django 的前后端分离项目

### 安装与运行

  - 后端
    - 基础环境：python3
    - 安装 Django
    ```
    pip3 install django
    ```
    - 运行 Django
    ```
    cd backend
    python3 manage.py runserver
    ```
  - 前端
    - 基础环境：Node
    - 安装依赖
    ```
    cd frontend
    npm install -g cnpm --registry=https://registry.npm.taobao.org
    cnpm i
    ```
    - 运行 Node 服务
    ```
    npm run dev
    ```
